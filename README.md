# 📒 Agenda Telefónica MVC

Proyecto desarrollado en **Java Swing** utilizando **Gradle** y el patrón de arquitectura **MVC (Modelo - Vista - Controlador)** como parte del **Hackathon 2 - Cohorte 12 Generation**.

## 📌 Descripción

La aplicación permite gestionar una agenda telefónica mediante una interfaz gráfica desarrollada con Java Swing.

El sistema permitirá:

- Agregar contactos.
- Buscar contactos.
- Modificar el número telefónico.
- Eliminar contactos.
- Listar contactos ordenados alfabéticamente.
- Validar datos de entrada.
- Evitar contactos duplicados.

## 🏗 Arquitectura

El proyecto sigue el patrón **MVC**.

### Modelo

Contiene la lógica de negocio y la gestión de los datos.

- Contacto
- Agenda

### Vista

Contiene la interfaz gráfica desarrollada con Java Swing.

- VentanaPrincipal

### Controlador

Gestiona la comunicación entre el Modelo y la Vista.

- AgendaControlador
